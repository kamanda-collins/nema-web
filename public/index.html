📄 FAQ – Professional Compliance Assessment Tool (Uganda)

❓ What does this tool do?

+

It checks if a set of coordinates falls within or near a wetland in Uganda, based on publicly available geospatial data. It then generates a professional-style output report in Word, Markdown, and PNG format to support Environmental Impact Assessment (EIA) processes.

❓ Who is this tool for?

+
  • Environmental consultants
  • EIA practitioners
  • Developers or planners working near sensitive areas
  • Researchers & students in environmental science, GIS, or planning

❓ What format is the output?

+

You receive:

  • A Word Document (.docx) with a structured report template
  • A Markdown file (for devs/researchers who use versioning tools)
  • A PNG map image showing the queried area and proximity to wetlands

❓ Do I need GIS or coding skills?

+

No. You just input coordinates or location data. The system processes everything for you.

❓ How accurate is the wetland data?

+

The tool uses a simplified, open-source wetland dataset. It is suitable for preliminary screening, not a substitute for an official ground-truthed report.

❓ Can this be used outside Uganda?

+

The current version is specific to Uganda. We're exploring expansion depending on demand and available data.

❓ Is it NEMA certified?

+

No. This is not an official NEMA tool. It is designed to help professionals and students prepare and pre-screen data before submitting formal documentation. It follows NEMA-style report formats.
